WPMoo is a micro OOP framework for building WordPress admin pages, fields, and metabox panels.

== Description ==
WPMoo provides fluent builders for Options pages, Metabox panels, and common field types. It ships minimal runtime with optional dev helpers.

== Installation ==
Install and activate like any plugin.
